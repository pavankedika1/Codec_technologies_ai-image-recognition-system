# 🖼️ AI Image Recognition System
## 📌 Overview
The AI Image Recognition System is a web-based application that allows users to upload images 
and receive real-time object classification predictions using Deep Learning.
This project leverages TensorFlow's pre-trained MobileNetV2 model along with OpenCV for image processing and Flask for the web interface.
---

## 🚀 Features
* Upload images through a web interface
* Real-time image classification
* Deep Learning-based object recognition
* Confidence score prediction
* User-friendly Flask web application
* Image preprocessing using OpenCV
* Pre-trained MobileNetV2 model
---

## ⚙️ Installation
### Clone Repository
```bash
git clone https://github.com/yourusername/ai-image-recognition-system.git
cd ai-image-recognition-system
```

### Install Dependencies
```bash
pip install flask tensorflow opencv-python pillow numpy
```

### Run Application
```bash
python app.py
```

---

## 🌐 Usage
1. Open the application in your browser.
2. Upload an image.
3. Click the Predict button.
4. View the predicted object and confidence score.
